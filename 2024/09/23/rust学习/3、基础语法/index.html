条件语句

rust中的判断严格限制为布尔类型，不同于其他语言的非0即真。

if - else if - else语句

这样经典的判断rust同样支持，但是不同于C语言，rust的每个if-else块必须使用{ }，不允许独立的语句；不同于go语言，rust允许{在单独的行上。

三目运算

可以使用一个参数来接收作用域中的返回值

match

在rust中match关键字用于实现一个类似于C++中的switch的的选择分支结构，常常和枚举类型进行联用。

既然是函数体，可以是允许有返回值的。可以在match添加参数来接收这个返回值。另外，其他情况使用下划线_表示。

if let语句

if let是只区分两种情况的match语句的”语法糖”，用于处理比较复杂的语句，对于枚举类依然适用。

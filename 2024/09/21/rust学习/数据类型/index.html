变量声明

let关键字

rust是强类型语言，使用let关键字声明变量，会自推导类型，也可以指定类型。

可变变量、不可变变量、常量

  • 可变变量允许直接重新赋值
  • 不可变变量，只能重影（shadowing），类似一种重新绑定
  • 常量不可变

基本数据类型

注释

严格来说不属于数据类型。rust的注释和C的相同。

整型

位长度 有符号 无符号
8-bit i8 u8
16-bit i16 u16
32-bit i32 u32
64-bit i64 u64
128-bit i128 u128
架构 isize usize
进制 例
十进制 12_345 或 12345
十六进制 0xff
八进制 0o77
二进制 0b1111_0000（推荐）
字节(只能表示 u8 型) b’A’

下划线的设计能方便程员计数，当然添加下划线也是可以的

浮点

有两种类型

  • f32：32位浮点
  • f64：64位浮点

布尔

使用bool关键字进行声明，有两个取值。

  • ture
  • false

在判断中，rust严格使用布尔类型，不同于其他语言的非零即真

字符

使用char关键字进行声明，rust的字符型占4字节，是Unicode，使用单引号。
